# 🛍️ Retail Sales EDA & Insight Report

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover insights about customer behavior, product performance, and sales trends.

## 📂 Dataset

* Retail transaction dataset
* Includes customer demographics, product category, and sales details

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Key Analysis

* Sales by product category
* Monthly sales trends (seasonality)
* Customer segmentation (age, gender)
* Top customers
* Correlation analysis

## 📈 Key Insights

* Sales are concentrated in a few categories
* Strong seasonal trends observed
* Specific customer segments drive most revenue
* High-value customers contribute significantly

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Run Jupyter Notebook:

   ```
   jupyter notebook
   ```

## 📄 Output

* Notebook with full EDA
* Visualizations (charts)
* One-page insight report

## 💡 Conclusion

This project demonstrates how data-driven insights can improve business decision-making in retail.
